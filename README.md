# [Yokurang's Personal Website](https://yokurang.com/)

Welcome to the repository for my personal website, built using Next.js, TypeScript, and Tailwind CSS. You can view the live website at [yokurang.com](https://yokurang.com/).

## Overview

This project serves as my personal portfolio and a hub for my writings, projects, and contact information. Whether you're a recruiter, fellow developer, or just passing by, I hope you find something of interest here!

## Features

- **Home:** An introduction to who I am and what I do.
- **About:** More in-depth information about my background and skills.
- **Resume:** My professional experiences and education.
- **Projects:** A showcase of projects I've worked on.
- **Posts:** A collection of my writings and thoughts.
- **Contact:** How to get in touch with me.

## Technologies Used

- **Next.js:** For server-rendered React applications.
- **TypeScript:** Strongly typed programming language that builds on JavaScript.
- **Tailwind CSS:** A utility-first CSS framework for rapidly building custom designs.
